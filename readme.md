# 1. Crew Manager

Build with Laravel and Vue (Vuex + Vue router)
Tailwind css

### 1.1. Install dependencies

$ composer install
$ yarn

### 1.1. Start server

$ php artisan serve

### 1.2. Watch files

$ yarn watch

### 1.3. Migrations

To generate database tables and seed the database:

$ php artisan migrate:fresh --seed
